# أيقونات نظام مركز المامون الطبي

## الملفات المطلوبة:

1. **logo.png** - شعار المركز (60x60 بكسل)
2. **favicon.ico** - أيقونة المتصفح (16x16, 32x32 بكسل)
3. **hospital.png** - أيقونة المستشفى (24x24 بكسل)
4. **marker-blue.png** - علامة خريطة زرقاء (32x32 بكسل)
5. **marker-green.png** - علامة خريطة خضراء (32x32 بكسل)
6. **marker-red.png** - علامة خريطة حمراء (32x32 بكسل)
7. **user-avatar.png** - صورة افتراضية للمستخدم (40x40 بكسل)

## إذا لم تكن الصور متوفرة:

يمكن استخدام أيقونات Font Awesome بدلاً من الصور المخصصة. النظام مصمم للعمل مع أو بدون الصور المخصصة.

## نصائح للصور:

- استخدم تنسيق PNG للشفافية
- احفظ الصور بحجم مضاعف للشاشات عالية الدقة
- استخدم ألوان متناسقة مع نظام الألوان (#2563eb - أزرق أساسي)
